# Conhecendo Versionamento GitHub

## O que é GitHub?
GitHub é uma plataforma online que hospeda projetos de desenvolvimento de software, utilizando o sistema de controle de versão Git para gerenciar o código e rastrear alterações. Ele funciona como uma rede social para desenvolvedores, permitindo colaboração, compartilhamento de projetos, e também serve como portfólio. 

## Como funciona o GitHub?
O GitHub funciona como uma plataforma online que hospeda projetos de código usando o sistema de controle de versão chamado Git. Ele permite que desenvolvedores armazenem, acompanhem e compartilhem seu trabalho em "repositórios" na nuvem, colaborem com outras pessoas e gerenciem as alterações no código ao longo do tempo. 

### Baseado no Git: 
O GitHub utiliza o Git, um sistema de controle de versão que rastreia as mudanças no código localmente no computador. 

### Repositórios remotos: 
O GitHub é a plataforma que hospeda esses projetos como um "repositório remoto" na nuvem, permitindo o compartilhamento e a colaboração em larga escala. 

### Fluxo de trabalho: 
O processo de trabalho padrão envolve:

- Criar um branch para trabalhar em uma nova funcionalidade ou correção. 
- Fazer commits para salvar as alterações no seu repositório local. 
- Abrir uma pull request para propor a fusão das suas alterações com o código principal e iniciar uma discussão sobre elas. 
- Mergear: a pull request após a aprovação para incorporar o novo código. 

### Ferramentas de colaboração: 
A plataforma oferece ferramentas como: 
- Issues (discussões encadeadas), 
- Revisões de código  
- GitHub Marketplace para integrar aplicativos. 

### Comunidade e portfólio: 
Funciona como uma rede social, conectando desenvolvedores e permitindo que usem o GitHub como um portfólio para mostrar seus projetos a recrutadores.

### Funcionalidades Principais
- Hospeda repositórios Git na nuvem
- Oferece interface web para gerenciar projetos
- Facilita colaboração entre desenvolvedores
- Oferece features como Pull Requests, Issues e Wiki

## Principais Concorrentes do GitHub são:

### 1. GitLab
- Similar ao GitHub com CI/CD integrado
- Oferece versão community edition gratuita
- Auto-hospedagem disponível

### 2. Bitbucket
- Focado em integração com outras ferramentas Atlassian
- Planos gratuitos com repositórios privados
- Forte integração com Jira e Trello

### 3. Azure DevOps
- Parte do ecossistema Microsoft
- Oferece além do Git, ferramentas de DevOps
- Integração com Azure Cloud

### 4. SourceForge
- Uma das primeiras plataformas do gênero
- Foco em software open source
- Oferece hospedagem de downloads
