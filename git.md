# Git - Controle de Versão

## O que é Git?
Git é um sistema de controle de versão distribuído usado para rastrear mudanças no código-fonte durante o desenvolvimento de software.

## Como surgiu?
O Git foi criado em 2005 por **Linus Torvalds**, o mesmo criador do Linux. Surgiu da necessidade de um sistema de controle de versão mais eficiente após problemas com o BitKeeper.
